Bot Parameters

  • market: Trading pair in format BASE_QUOTE (e.g., BTC_USDT)
  • exchange: Exchange to copy order book from (e.g., binance, kraken)
  • min_orders: Minimum number of orders to maintain
  • trade_amount_min: Minimum amount for each trade
  • trade_amount_max: Maximum amount for each trade
  • frequency_from: Minimum time between actions (seconds)
  • frequency_to: Maximum time between actions (seconds)
  • price_factor: Price deviation percentage
  • market_gap: Market gap percentage between best price and placed orders
